# Network Scanner (فاحص الشبكة)

أداة بسيطة بلغة Python لفحص الشبكة المحلية وعرض الأجهزة المتصلة من خلال إرسال أوامر ping.

## طريقة التشغيل

```bash
python scanner.py
```

ثم أدخل نطاق الشبكة، مثال:
```
192.168.1
```

## المتطلبات

- Python 3
- لا حاجة لأي مكتبة خارجية

## الاستخدامات

- التعرف على الأجهزة المتصلة بالشبكة
- مراقبة نشاط الشبكة
